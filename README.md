# BakeryOS

BakeryOS is a modern bakery management system.

## Project Structure

- `apps/`: Web and mobile applications.
- `packages/`: Shared libraries and components.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run in development mode:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
